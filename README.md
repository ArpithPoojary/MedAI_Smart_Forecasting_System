# MedAI Smart Forecasting System

MedAI Smart Forecasting System is an AI-powered pharmacy inventory management and medicine demand forecasting web application.

The system helps pharmacies analyze previous sales data, monitor inventory, predict future medicine demand, and reduce medicine wastage caused by overstocking or shortages.

---

## Features

- User Authentication & Authorization
- Admin and User Dashboard
- Medicine Sales Dataset Upload
- Data Cleaning & Preprocessing
- Medicine Demand Forecasting
- Stock Monitoring
- Sales Analytics & Reports
- Overstock and Restock Alerts
- Graphical Dashboard Visualization
- OTP-based Password Recovery

---

## Technologies Used

### Frontend
- React.js
- Tailwind CSS
- JavaScript
- Vite

### Backend
- FastAPI
- Python

### Machine Learning
- Scikit-learn
- Pandas
- NumPy

### Database
- SQLite

---

## Project Structure

```bash
backend/
frontend/
```

### Backend

```

```bash
